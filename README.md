🍔 McDonald’s Business Performance Dashboard

📌 Overview : 
This project analyzes McDonald’s operational and sales data to identify performance gaps, improve customer experience, and support data-driven decision-making. The analysis focuses on revenue trends, service efficiency, staffing optimization, and customer satisfaction.

⚙️ Tools & Technologies : 
Excel, Python, Power BI

📊 Key Features
1) Multi-Dataset Integration - 
Combined sales, branch, and staffing datasets for comprehensive analysis
2) KPI Development & Tracking - 
Designed key metrics 
3) Branch Performance Analysis - 
Identified unstable revenue trends and underperforming branches
4) Customer Behavior Insights - 
Analyzed preference for drive-thru and delivery channels
5) Operational Efficiency Analysis - 
Evaluated staffing levels and identified diminishing returns
6) Customer Satisfaction Analysis - 
Detected high complaint rates and key dissatisfaction drivers
7) Interactive Power BI Dashboard - 
Built a dashboard for real-time monitoring and decision support
